# 🚀 WellMind AI

Track habits. Analyze patterns. Improve your life with AI.

WellMind AI is an AI-powered habit and health tracking system that helps users monitor daily routines, identify behavioral patterns, and receive personalized insights to improve productivity and well-being.

---

## ✨ Features

- Track daily habits (sleep, water, mood, activity)
- AI-based insights and suggestions
- Analyze patterns in daily routines
- Simple and clean dashboard
- Future: predictive analytics & personalized recommendations

## 🚀 Getting Started

git clone https://github.com/radhika-droid/wellmind-ai-.git
cd wellmind-ai-/backend

python3 -m venv venv
source venv/bin/activate

pip install -r ../requirements.txt

python manage.py runserver
